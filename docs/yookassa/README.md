и мы ожидаем выдачи API-ключей. На них ссылается корневой `README.md`.

| Файл | Что на нём |
|---|---|
| `moderation_status.png` | Карточка магазина в личном кабинете ЮKassa: `https://finaudit.site`, ShopID 1406623, статус «Подключается» |
| `support_chat.png` | Ответ поддержки ЮKassa: магазин 1406623 на проверке, ключи API пока недоступны, подключение до 2 календарных дней |

Перед публикацией убедитесь, что на скриншотах нет секретных данных
(ключей API, паролей, платёжных реквизитов).
